# Akhil Verma - Frontend Engineering Portfolio

Welcome to the source code for my professional portfolio website!

## 🚀 Overview
This portfolio is built with modern web technologies to showcase my professional experience, projects, technical skills, and achievements. It focuses heavily on performance optimization, clean UI/UX, and responsive design.

## 🛠 Tech Stack
- **Core:** React, TypeScript, Vite
- **Styling:** Vanilla CSS, Tailwind CSS Utility Concepts
- **Icons:** React Icons
- **Contact Form:** Web3Forms Integration

## 📦 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository
```bash
git clone https://github.com/Akhil11V/portfolio.git
```

2. Navigate into the directory
```bash
cd portfolio
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```
